Data Augmentation for Brain Tumor Detection

Explore how data augmentation enhances our dataset, improving model performance and addressing class imbalance.

Introduction to Data Augmentation

Data augmentation is a crucial step in preparing our dataset for training robust machine learning models. By artificially expanding the dataset, we can improve the model's ability to generalize and reduce overfitting. In this project, we employ a diffusion model to generate synthetic images, thereby enhancing the diversity and balance of our training data.

Diffusion Model for Synthetic Image Generation

The diffusion model is a generative approach that creates new data points by learning the underlying distribution of the original dataset. This model is particularly effective in generating realistic and varied images, which are essential for training deep learning models.

Key steps in the diffusion model process include:

  • Training the Model: The diffusion model is trained on the original dataset to learn the distribution of each tumor category.
  • Generating Images: Once trained, the model generates synthetic images that mimic the characteristics of the original data.
  • Integration: These synthetic images are integrated into the training dataset, making up 25% of the final dataset.

Impact on Dataset

The integration of synthetic images significantly impacts the dataset by:

  • Balancing Classes: Augmentation helps to address class imbalance, ensuring that each tumor category is well-represented.
  • Enhancing Diversity: The synthetic images introduce variability, which is crucial for improving the model's generalization capabilities.
  • Increasing Dataset Size: The final training dataset consists of 7616 images, providing a robust foundation for model training.

Augmentation Metrics

Below is the distribution of images in each category after augmentation:

  • Glioma Tumors: 2000 images (1500 original, 500 augmented)
  • Meningioma Tumors: 1700 images (1300 original, 400 augmented)
  • Pituitary Tumors: 1500 images (1200 original, 300 augmented)
  • Non-Tumor Cases: 2416 images (1712 original, 704 augmented)

These metrics highlight the effectiveness of the augmentation process in creating a balanced and comprehensive dataset.
